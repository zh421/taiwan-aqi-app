# 簡介
登入前需要先註冊，註冊資訊寫在info/credentials.txt  
註冊後第一次進入會前往設定頁面，需要輸入api，api資訊寫在info/api.txt。輸入完成後的每次登入，會自動前往查詢頁面。
  
查詢頁面有三個項目查詢：空氣品質、紫外線、沙塵  
空氣品質有兩項查詢方式：
 - 依測站進行查詢
 - 依空氣物質進行查詢
